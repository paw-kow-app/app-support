# App Support Website

This folder contains a small static website for App Store submission and future app support pages.

- `index.html`: central landing page for all apps
- `assets/styles.css`: shared website styling
- `apps/corner-bell/index.html`: app-specific landing page
- `apps/corner-bell/privacy.html`: public privacy policy URL
- `apps/corner-bell/support.html`: public support URL
- `apps/corner-bell/legal.html`: Apple Standard EULA and safety information

Support email: `paw.kow.app@icloud.com`

Suggested GitHub Pages URLs after hosting the repository as `app-support`:

- Main site: `https://paw-kow-app.github.io/app-support/`
- App page: `https://paw-kow-app.github.io/app-support/apps/corner-bell/`
- Privacy Policy URL: `https://paw-kow-app.github.io/app-support/apps/corner-bell/privacy.html`
- Support URL: `https://paw-kow-app.github.io/app-support/apps/corner-bell/support.html`
- Legal URL: `https://paw-kow-app.github.io/app-support/apps/corner-bell/legal.html`

The site is static and does not need a backend, build process or external dependencies.

Before App Store submission, verify whether your jurisdiction requires additional public legal notice information such as a postal address or phone number. Do not publish placeholder contact details.
