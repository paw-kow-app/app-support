# App Store Website

This folder contains a small static website for App Store submission:

- `index.html`: overview page
- `privacy.html`: public privacy policy URL
- `support.html`: public support URL
- `styles.css`: shared styling

Support email: `paw.kow.app@icloud.com`

Suggested App Store Connect URLs after hosting:

- Privacy Policy URL: `https://YOUR_DOMAIN_OR_HOST/privacy.html`
- Support URL: `https://YOUR_DOMAIN_OR_HOST/support.html`

The site is static and does not need a backend, build process or external dependencies.
